# Build & Run
## Install Requirements
The software you need to install is: **qemu & nasm** <br>
To install them in linux run
```bash
sudo apt install qemu nasm
```
## Building and running
Run in the root of the repo:
```bash
chmod +x ./run.sh
./run.sh
```
